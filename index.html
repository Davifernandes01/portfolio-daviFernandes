Projeto feito para o challenge 2 do projeto ONE, nesse projeto desenvolvemos do zero um jogo da forca, que teria que aceitar palavras digitadas pelo usuário. nesse projeto usei uma função nova para mim, que foi a "split(" ")", utilizando ela para separar as letras da palavra escolhida em um array, meus maiores desafios nesse jogo foi, guardar a palavra digitada pelo usuario e usa-lá em outra pagina, que nesse caso, utilizei o LocalStorage para armazena-lá. O outro foi para desenhar os pontos de cada letra independente da palavra escolhida, que nesse caso utilizei um loop for.


Esse foi o primeiro challenge do projeto ONE e meu primeiro projeto publicado no github, nesse projeto fizemos um encriptador e desencriptador de textos, aonde ao digitar uma palavra, aonde tiver as vogais A, E , I , O ,U , será trocadas essas vogais e será adicionada no lugar as palavras "ai", "enter", "imes", "ober", "ufat", respectivamente. quando o usuario apertar no botão "descriptografar", acontecerá o contrario, aonde estiver escrito, "ai", "enter", "imes", "ober", "ufat" será trocado pelas vogais, A, E , I , O ,U. foi um projeto bem simples, mas a parte dificil dele foi entender a função "replace()".


Nesse projeto feito em um curso de javascript do projeto ONE, fizemos todo o javascript dessa pagina, com o auxilio do instrutor do curso, fizemos funções(muitas), aprendemos um pouco de POO (programação orientada a objetos), aprendemos a como consumir uma API da web, utilizamos AJAX, para atualizar a pagina as informações da pagina automaticamente, manipulamos o DOM e outras varias atividades passadas pelo instrutor.


Esse foi o primeiro projeto que fiz no curso de html e css, nesse curso o instrutor nos ensinou o basico sobre o html como por exemplo o uso correto de tags, o basico de css como display, color, background-color, margin, position, etc, foi um curso bem explicativo e tenho certeza que aprendi muito com ele.
